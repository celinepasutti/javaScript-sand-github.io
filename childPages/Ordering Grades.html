Ordering Grades Assignment

Description of Assignment
  • Problem: Calculate up to 5 test scores into a letter value.
  • Test cases:
    • F: 1-49%
    • D: 50-59%
    • C: 60-69%
    • B: 70-79%
    • A: 80-100%
  • Note: 0% is not allowed.
  • Note: calculating means adding 5 test scores and dividing by the number of scores.
  • Note: able to have less than 5 test scores and mean should still work.
  • Note: your code must check if there is spaces or textboxes left blank.
