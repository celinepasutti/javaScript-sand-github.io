Demerit Points Assignment

Description of Assignment
  • Problem: input a speed and calculate the demerit points
  • Bonus (Advanced): also input the speed limit
  • Note: speed limit is 70km
  • Note: for every 5km above the speed limit
  • Note: at any speed limit, 12 demerit points = license suspended
  • Test cases:
    • 68km/h: 0 points
    • 70km/h: 0 points
    • 71km/h: 0 points
    • 74km/h: 0 points
    • 75km/h: 1 points
    • 85km/h: 3 points
    • 102km/h: 6 points (round down)
    • 115km/h: 9 points
    • 130km/h: lisence suspended
  • Coding Hints:
    • math.floor(): function to remove any decimal places, never rounding.
